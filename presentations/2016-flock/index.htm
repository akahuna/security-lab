What's Fedora Security Lab?

A safe test environment to work on security auditing, forensics, system rescue, and teaching security testing methodologies.


The Fedora Security Lab is an official spin.

Fedora Security Lab features

  • Live CD
  • Runs from USB devices
  • Xfce Desktop Environment
  • over 100 specialized tools
  • customized menu
  • installable

History

  • included since Fedora 13
  • in the beginning OpenBox, then LXDE
  • constant developement

Coverage

  • Reconnaissance
  • Forensics & Rescue
  • Intrusion detection
  • Network Analysis
  • Wireless
  • Password Tools
  • Code Analysis
  • Web Application Testing
